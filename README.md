# Sales Insights Dashboard

## Project Overview
This project analyzes sales performance using Power BI. The dashboard provides insights into revenue, profit, customer behavior, and regional sales trends.

## Objectives
- Monitor sales performance
- Identify top-performing products
- Analyze customer and market trends
- Track revenue and profit KPIs

## Tools Used
- Power BI
- SQL
- Excel

## Key KPIs
- Total Revenue
- Total Profit
- Sales Quantity
- Top Customers
- Top Products

## Dashboard Features
- Revenue Trend Analysis
- Market-wise Sales Analysis
- Product Performance Analysis
- Customer Contribution Analysis

## Files Included
- Power BI Dashboard (.pbix)
- Dataset (.sql)
- Dashboard Screenshots
